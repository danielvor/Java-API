## Descrição

Esta API permite criar, listar, buscar, atualizar e deletar tarefas. Cada tarefa possui um título e um conteúdo.

## Tecnologias Utilizadas

- Java 17
- Spring Boot 3.5.0
- Spring Data MongoDB
- MongoDB
